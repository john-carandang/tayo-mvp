B. Stream Claude responses sentence by sentence:

Instead of waiting for the full response before calling ElevenLabs, split the response into sentences as they arrive and send each sentence to ElevenLabs as soon as it ends with ., ?, or !. Queue the resulting audio chunks and play them sequentially. This means audio begins playing after the first sentence arrives rather than after the full response is complete.
Implementation approach:
javascript// In /api/chat — stream Claude, collect sentences, speak each
const stream = await anthropic.messages.stream({ ... });
let buffer = '';
let audioQueue = [];
let isPlaying = false;

stream.on('text', async (text) => {
  buffer += text;
  const sentences = buffer.split(/(?<=[.?!])\s+/);
  if (sentences.length > 1) {
    const toSpeak = sentences.slice(0, -1).join(' ');
    buffer = sentences[sentences.length - 1];
    // Send toSpeak to ElevenLabs, queue audio
    audioQueue.push(await generateAudio(toSpeak));
    if (!isPlaying) playNext();
  }
});
stream.on('end', async () => {
  if (buffer.trim()) {
    audioQueue.push(await generateAudio(buffer));
    if (!isPlaying) playNext();
  }
});
